
# NewsApp

iOS client for [NewsAPI.org](https://newsapi.org) based on Swift.


## Installing

- Visit [NewsAPI.org](https://newsapi.org) to get your API Key
- Paste the key in APIConstants.swift file.

```swift
static let apiKey = "API"
```

## Built With

- SwiftUI
- CoreData
- LottieFiles
